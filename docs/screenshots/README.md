# Screenshots for the project README

Drop the following PNG files in this directory and they'll appear in the
top-level `README.md`:

- `menubar.png` — the menubar dropdown with quick controls
- `settings.png` — the Settings window with the 5 tabs

## How to capture

1. Run the app: `swift run DualSenseMac` (or run from Xcode)
2. **Menubar shot:** click the gamecontroller icon in the menubar so the
   dropdown is open. Press `Shift-Cmd-4`, then press `Space`, then click
   the menubar dropdown window. Save the screenshot, rename to `menubar.png`,
   move into this directory.
3. **Settings shot:** in the menubar dropdown click "Open Settings…" so
   the window appears. Press `Shift-Cmd-4`, then `Space`, then click the
   Settings window. Save and rename to `settings.png`.
